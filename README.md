Ashes of Alengka is a local multiplayer shadow puppet fighting simulator based off of the Ramayana story. The aim is to spread traditional Javanese/Malay/Indian tradition in a fun way.

Each player holds half of the controller. The left analog stick controls Anoman (The White Monkey), while the right analog stick controls Dasamuka (The Ten Faced King). The characters move by tilting, just like an actual wayang being held by their sticks. Players can either lean left or right to approach and avoid the oponent. Players can also swing an attack with the dpad or the ABXY buttons in controller. 

The winner is the one who drains the last drop of their opponent's health.

How to run the project
1. Install Godot version 4.6.2 {link}
2. Clone the repository
3. Open the project in Godot
4. It is advised to connect a game controller for better play
5. Have fUN!
